md2static
This project is a Go implementation of John Gruber's Markdown. I'm currently working on it to develop a grammar for Markdown using the ANTLR parser generator. My initial focus is on contributing a clean, reusable Markdown grammar back to the community, as I've benefited from grammars shared in the antlr/grammars-v4 repository.
The long-term goal of this project is to evolve it into a static site generator, similar to MkDocs, built entirely in Go.

I'm currently in the process of writing detailed descriptions for my projects.

In the meantime, feel free to explore my work directly on GitHub . You'll find a variety of repositories there that showcase what I've been building.
